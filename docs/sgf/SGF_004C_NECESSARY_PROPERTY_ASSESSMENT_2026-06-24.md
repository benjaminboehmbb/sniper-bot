# Purpose

Evaluate whether each recognized Candidate Property represents a Necessary Property of the scientific object.

The assessment shall determine necessity only.

It shall not perform Necessary Property Recognition.

---

# Scientific Question

Does the available validated scientific evidence require the recognized Candidate Property to be treated as a Necessary Property of the scientific object?

---

# Inputs

Accepted inputs:

- SGF-003H Candidate Property Recognition
- SGF-004B Necessary Property Investigation

Forbidden inputs:

- implementation assumptions
- architectural assumptions
- optimization concepts
- domain-specific reasoning
- future applications

---

# Assessment Criteria

Every Candidate Property shall be evaluated using:

- explanatory necessity,
- indispensability,
- evidence sufficiency,
- universality within the validated evidence,
- traceability.

No criterion shall rely on implementation usefulness.

PASS
