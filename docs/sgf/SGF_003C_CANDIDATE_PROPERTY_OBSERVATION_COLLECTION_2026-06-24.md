# Purpose

Collect observable properties of the previously recognized scientific object.

No Necessary Property shall be introduced, assessed or recognized during this phase.

---

# Scientific Question

Which observable properties of the recognized scientific object can be collected without introducing scientific interpretation?

---

Accepted inputs:

- Validated SGF-002B
- Validated SGF-003B

Forbidden inputs:

- Necessary Property definitions
- property classifications
- implementations
- architecture
- optimization concepts
- application-specific assumptions

# Observation Rules

Every observation shall be:

- directly observable
- interpretation-independent
- reproducible
- formulation-independent where practical

No observation may state that a property is necessary.
