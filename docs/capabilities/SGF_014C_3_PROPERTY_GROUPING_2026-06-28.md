# Purpose

The purpose of this document is to organise the classified scientific properties into coherent Scientific Property Groups.

Property Groups represent stable scientific structures consisting of multiple related Property Classes.

Property Grouping does not derive Property Configurations.

Property Grouping does not recognise Scientific States.

Property Grouping does not derive Scientific Forecast Representation.

Its purpose is solely to identify higher-order scientific structures emerging from combinations of previously classified scientific properties.
