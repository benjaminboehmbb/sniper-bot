# Purpose

The purpose of this document is to organise the classified Scientific Property Configurations into higher-order Scientific Property Configuration Groups.

Property Configuration Groups represent stable scientific organisations of recurring Property Configuration Classes.

Property Configuration Grouping does not recognise Scientific State Candidates.

Property Configuration Grouping does not recognise Scientific States.

Property Configuration Grouping does not derive Scientific Forecast Representation.

Its purpose is solely to identify higher-order scientific structures emerging from recurring Property Configuration Classes.
