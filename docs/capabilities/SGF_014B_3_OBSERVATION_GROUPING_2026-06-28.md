# Purpose

The purpose of this document is to organise the classified scientific observations into coherent scientific observation groups.

Observation Groups represent higher-order scientific structures composed of multiple related observation classes.

Grouping does not derive scientific properties.

Grouping does not introduce scientific entities.

Grouping does not interpret scientific meaning.

Its sole purpose is to identify stable scientific observation structures that repeatedly occur across the collected evidence.
