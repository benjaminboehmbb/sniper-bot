# Purpose

The purpose of this document is to classify the scientific properties identified during SGF-014C.1.

Property Classification does not group properties.

Property Classification does not derive property configurations.

Property Classification does not recognise scientific states.

Its sole purpose is to organise recurring scientific properties into scientifically distinguishable property classes.

The resulting classification shall prepare Property Grouping during the following derivation phase.
