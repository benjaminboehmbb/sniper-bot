# Purpose

The purpose of this document is to identify recurring Scientific Property Configurations emerging from the previously identified Property Groups.

A Scientific Property Configuration represents a stable recurring combination of scientific properties.

Property Configuration Identification does not classify Property Configurations.

Property Configuration Identification does not recognise Scientific States.

Property Configuration Identification does not derive Scientific Forecast Representation.

Its sole purpose is to identify recurring combinations of scientific properties consistently supported by the collected scientific evidence.
