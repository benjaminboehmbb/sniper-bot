# Purpose

The purpose of this document is to classify the recurring Scientific Property Configurations identified during SGF-014D.1.

Property Configuration Classification organises recurring configuration types.

It does not derive Property Configuration Groups.

It does not recognise Scientific State Candidates.

It does not recognise Scientific States.

Its purpose is solely to establish scientifically distinguishable categories of recurring Property Configurations.
