# SGF-013G – Necessary Property Validation

---

# 1. Objective

The objective of this phase is to validate whether each candidate Necessary Property satisfies the Scientific Necessary Property criteria.

Validation determines whether every property is:

* scientifically necessary,
* capability-specific,
* independently justified,
* non-redundant,
* consistent with the candidate scientific responsibility.

Only validated properties may become part of the final Property Configuration.

---

# 2. Validation Criteria

Each Necessary Property is evaluated according to the following criteria.

* Necessity
* Capability specificity
* Scientific traceability
* Independence from other properties
* Internal consistency
