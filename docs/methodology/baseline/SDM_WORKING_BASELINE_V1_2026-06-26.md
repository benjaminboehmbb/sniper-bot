# 1. Objective

The Scientific Derivation Methodology (SDM) defines a scientific process for deriving new scientific capabilities.

The methodology ensures that new scientific capabilities are introduced only through explicit scientific justification, traceable evidence, and minimal necessary structure.

---

# 2. Scientific Principles

Every scientific derivation follows these principles:

- Scientific Justification First
- Observation Before Conclusion
- Explicit Traceability
- Scientific Minimality
- Single Scientific Responsibility
- Dependency-Driven Evolution
- Separation of Observation and Assessment
- Separation of Scientific Derivation and Implementation

---

# 3. Governance Principles

The current Working Baseline uses the following governance checks:

- Functional Requirement Analysis
- Capability Gap Analysis
- Scientific Dependency Analysis
- Scientific Problem Analysis
- Architecture Evolution Review
- Counterfactual Review
- Removal Test
- Compression Test
- Final Scientific Review
- Methodology Review after multiple capability pilots

Methodological changes require repeated evidence.
