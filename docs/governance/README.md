# Scientific Governance

This directory contains the platform-wide Scientific Governance documentation.

Scientific Governance defines normative standards, reviews, certifications, registries, and change history for scientific governance activities.

## Structure

- `standards/` contains official Scientific Governance Standards.
- `reviews/` contains governance reviews.
- `certifications/` contains formal certification records.
- `changelog/` contains version and release change records.
- `registry/` contains governance standard registries.

## Current Standards

- SCG_001: Scientific Compute Governance V1
