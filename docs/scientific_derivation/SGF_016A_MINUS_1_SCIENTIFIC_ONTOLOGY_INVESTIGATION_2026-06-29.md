# Status

DRAFT - ONTOLOGICAL INVESTIGATION

# Scientific Question

What is the ontological status of Scientific Evaluation?

The term "Evaluation" is treated throughout this investigation as a candidate scientific concept.

Its ontological existence shall not be assumed.

One objective of this investigation is to determine whether Evaluation exists as an independent scientific concept or whether it can be reduced to already existing scientific structures.

# Purpose

This investigation determines whether the candidate concept "Evaluation" is a valid candidate for an independent Scientific Core capability.

This document does not define SGF-016.

This document does not certify SGF-016.

This document only investigates whether Evaluation has an ontological status that justifies further capability derivation.

# 1. Initial Observation

SGF-015 introduced Scientific Forecast Evaluation Representation.

SGF-015 represents evaluation of Forecast Representations.

This creates an unresolved question:

Is Forecast Evaluation a fundamental capability, or is it a specialized instance of a more general candidate concept currently referred to as "Evaluation"?

# 2. Ontological Problem

Previous Scientific Core capabilities primarily introduced or refined scientific representations of objects, configurations, states, behaviours and forecasts.

Evaluation differs from these prior structures.

Evaluation appears to be a scientific statement about another scientific object.

Therefore, Evaluation may belong to a different ontological level.

# 3. Competing Hypotheses

## H1 - Evaluation as Scientific Object

Scientific Evaluation is an independent scientific object class.

If true, SGF-016 may be justified as Scientific Evaluation Representation.

## H2 - Evaluation as Relation

Scientific Evaluation is not an object.

It is a relation between an evaluator, an evaluated object and an evaluation basis.

If true, SGF-016 may require a Scientific Relation Representation instead.

## H3 - Evaluation as Meta-Statement

Scientific Evaluation is a scientific meta-statement about a scientific object.

If true, SGF-016 may require a more general Scientific Meta-Statement Representation.

## H4 - Evaluation as Process Result

Scientific Evaluation is not a representation capability.

It is the result of a later reasoning, validation or decision process.

If true, SGF-016 must not be introduced yet.


## H5 - Evaluation Does Not Exist as an Independent Scientific Concept

Evaluation is not an independent scientific concept.

Instead, the term "Evaluation" is merely a linguistic abstraction summarizing combinations of already existing scientific concepts such as observations, properties, states, behaviours, forecasts, relations or scientific statements.

If this hypothesis is supported, Scientific Evaluation Representation shall not be introduced as a new Scientific Core capability.

Instead, the apparent need for Evaluation Representation shall be explained entirely by previously established scientific structures.

# 4. Scientific Evaluation Criteria

Each hypothesis shall be evaluated using the same scientific criteria.

For every hypothesis the following shall be identified:

- Supporting scientific observations
- Contradicting scientific observations
- Scientific strengths
- Scientific weaknesses
- Acceptance criteria
- Rejection criteria

A hypothesis shall only remain scientifically valid if it cannot be falsified by the available scientific evidence.

# 4.1 Reduction Test

Evaluation cannot be reduced to Observation alone.

Observation represents what is scientifically observed.

Evaluation expresses an assessment of scientific adequacy, relation, fit, reliability, sufficiency or comparative value.

Evaluation cannot be reduced to Property alone.

A Property characterizes a scientific object.

Evaluation assesses an object or representation against criteria or reference conditions.

Evaluation cannot be reduced to State alone.

A State represents a scientific condition.

Evaluation expresses a judgement-like scientific relation or statement about scientific information.

Evaluation cannot be reduced to Forecast Evaluation alone.

Forecast Evaluation applies only to Forecast Representations.

Scientific Evaluation may apply to non-forecast scientific objects.

# 5. Process Separation Test

Evaluation execution and Evaluation Representation must be separated.

The Scientific Core may represent evaluation information.

It must not perform evaluation, compute scores, optimize choices or execute decisions.

Therefore, if Evaluation is accepted, it can only be accepted as Evaluation Representation.

# 6. Generalization Test

Forecast Evaluation appears to be a specialized form of a broader pattern:

Evaluation(Object)

Possible evaluated objects include:

- Observation
- Property
- Property Configuration
- Scientific State
- Scientific Behaviour
- Scientific Forecast
- Scientific Capability
- Scientific Architecture
- Scientific Derivation

This supports the hypothesis that Forecast Evaluation is not the most general form.

Before assessing the competing hypotheses, the ontological categories themselves shall be distinguished.

For the purpose of this investigation, the following candidate categories are recognized:

- Scientific Object
- Scientific Relation
- Scientific Statement
- Scientific Representation
- Scientific Process
- Scientific Result

These categories shall not be treated as equivalent.

One objective of this investigation is to determine to which category the candidate concept "Evaluation" belongs.

Generalization alone does not justify introduction of a new Scientific Core capability.

A generalized concept shall only be accepted if it introduces an irreducible scientific capability that cannot be represented by existing Scientific Core structures.

# 7. Ontological Level Assessment

Evaluation is not currently observed to behave like a first-order scientific object.

Based on the observations collected in this investigation, H1 currently appears less consistent than the competing hypotheses.

However, H1 cannot yet be rejected because no formal falsification has been completed.

Evaluation has relational structure.

Therefore, H2 remains plausible.

Evaluation has meta-statement structure.

Therefore, H3 remains plausible.

Evaluation may be produced by reasoning or validation processes.

However, the representation of the result can still exist independently of the process.

Therefore, H4 is partially valid but does not eliminate Evaluation Representation.

# 8. Provisional Finding

The strongest current interpretation is:

Scientific Evaluation is a structured scientific meta-statement with relational structure.

It is not merely a Forecast-specific capability.

It is not an execution process.

It is not a decision.

It is not selection.

It is not reasoning.

# 9. Consequence for SGF-016

SGF-016 remains scientifically plausible.

However, its provisional title may require refinement.

Possible names:

- Scientific Evaluation Representation
- Scientific Evaluation Statement Representation
- Scientific Meta-Evaluation Representation

The next document must determine the minimal capability gap.

# 10. Constraints for Next Step

The next SGF-016 document must test:

1. whether Evaluation requires a distinct representation,
2. whether Meta-Statement Representation is more fundamental,
3. whether Relation Representation is required first,
4. whether SGF-015 must remain unchanged,
5. whether SGF-016 can be introduced without retroactively invalidating SGF-015.

# 11. Investigation Result

The present investigation does not establish that Scientific Evaluation exists as an independent scientific concept.

Instead, the investigation establishes that the ontological status of the candidate concept "Evaluation" remains unresolved.

The following hypotheses remain scientifically plausible:

- Evaluation as Scientific Relation
- Evaluation as Scientific Meta-Statement
- Evaluation as Scientific Representation
- Evaluation as a non-independent linguistic abstraction

At the current stage, no hypothesis has been sufficiently supported or falsified.

Therefore, no ontological conclusion shall be drawn.

Instead, the subsequent Capability Gap Analysis shall continue the investigation by attempting to falsify each remaining hypothesis.

Only after successful falsification of competing explanations may an independent Scientific Evaluation Representation be introduced as a Scientific Core capability.

# Final Result

The ontological investigation successfully identified unresolved scientific questions regarding the candidate concept "Evaluation".

No independent Scientific Core capability has been justified at this stage.

The investigation therefore recommends continuation of scientific falsification before capability derivation.

The investigation produced a reusable scientific methodology for assessing the ontological status of future candidate Scientific Core capabilities.

Future capability derivations should first determine the ontological status of a candidate concept before introducing a new Scientific Core capability.
