# Definition of a Scientific Property

Within the Scientific Core,

a Scientific Property is a minimal scientifically distinguishable characteristic that contributes to the scientific representation, description or differentiation of a scientific object or a scientifically relevant aspect of that object.

A Scientific Property shall not:

- represent a scientific state;
- represent scientific behaviour;
- represent a scientific capability;
- describe implementation;
- describe algorithms;
- describe application-specific behaviour.

Scientific Properties therefore represent the minimal descriptive building blocks from which later Scientific Core concepts may be derived.
