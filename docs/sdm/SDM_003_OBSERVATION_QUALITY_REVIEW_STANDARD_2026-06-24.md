# Purpose

Define a standardized methodology for evaluating scientific observations before any interpretation, analysis or derivation is performed.

The objective of OQR is to ensure that accepted observations are maximally objective, minimally interpretative and reproducible.

---

# Scientific Question

What minimum quality requirements must a scientific observation satisfy before it may enter the derivation process?

---

# Scope

This standard applies to every Characteristics Collection performed using SDM, including future investigations of:

- Constraint
- Necessary Property
- Minimal Structure
- Architecture
- any future scientific object class

---

# Observation Quality Review Workflow

Every observation shall be evaluated in the following sequence.

---

## OQR-1

### Direct Observability

Question:

Can the statement be directly observed?

Decision:

PASS / FAIL

---

## OQR-2

### Interpretation Independence

Question:

Does the statement avoid interpretation, explanation or theoretical assumptions?

Decision:

PASS / FAIL

---

## OQR-3

### Observation Minimality Test (OQR-M1)

Question:

Can the same observation be expressed using fewer theoretical assumptions while preserving the observed phenomenon?

Decision:

PASS / FAIL

Reviewer Principle:

Reduce theoretical content until further reduction would remove the observation itself.

---

## OQR-4

### Reproducibility

Question:

Can independent reviewers reproduce the same observation using the same evidence?

Decision:

PASS / FAIL

---

# Observation Status Categories

Each reviewed observation shall receive one status.

## PASS

Accepted without modification.

---

## PASS WITH REVISION

Scientific content accepted.

Formulation revised to improve objectivity or reduce interpretation.

---

## UNDER REVIEW

Insufficient evidence for a decision.

---

## FAIL

Observation violates one or more OQR criteria.

Revision required.

---

## REJECTED

Observation removed from the derivation process.

---

# Observation Minimality Principle

Scientific observations shall always be expressed using the weakest formulation that fully preserves the observable phenomenon.

Whenever two formulations describe the same observation, the formulation containing fewer theoretical assumptions shall be preferred.

---

# Relationship to SDM

Observation Quality Review is performed before:

- Characteristics Analysis
- Object Assessment
- Scientific Investigation
- Scientific Derivation

OQR therefore represents the quality gate between raw observations and scientific reasoning.

---

# Acceptance Criteria

This standard is complete only if:

- every observation passes OQR before analysis,
- all four review stages are applied,
- observation status is explicitly documented,
- theoretical assumptions are minimized.
