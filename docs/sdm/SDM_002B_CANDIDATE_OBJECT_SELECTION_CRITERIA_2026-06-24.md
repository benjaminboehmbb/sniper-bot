# Purpose

Derive objective criteria for selecting scientific object classes suitable for cross-application validation of SDM.

No candidate object shall be selected before the selection criteria have been justified.

---

# Scientific Question

Which properties must a scientific object possess to provide meaningful evidence for validating the general applicability of SDM?

---

# Inputs

Accepted inputs:

- SDM-001 Meta Architecture Review
- SDM-002A Methodology Cross-Application Validation Investigation

Forbidden inputs:

- preferred object classes
- implementation convenience
- expected outcomes
- application-specific priorities

---

# Candidate Selection Criteria

A suitable candidate object should:

- represent a distinct epistemic object class
- differ meaningfully from previously validated object classes
- be derivable using SDM without modifying SDM
- exercise multiple SDM dimensions
- permit objective validation
- contribute additional methodological evidence

---

# Acceptance Criteria

This document is complete only if:

- the criteria are solution-neutral
- the criteria are domain-independent
- no candidate objects have yet been selected
- the criteria justify later candidate selection
