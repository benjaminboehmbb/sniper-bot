# Validation Criteria

The validation shall determine whether:

- Scientific Object classification remains unambiguous.
- Scientific Lifecycle remains sufficient.
- Review Perspectives remain sufficient.
- No additional SDM dimension becomes necessary.
- Existing derivation principles remain applicable.
- Existing review methodology remains applicable.
